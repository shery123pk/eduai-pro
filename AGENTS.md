# EduAI Pro Agent Instructions

## Mission
Build a production-ready AI Education Platform for BWAI Hackathon 2026

## Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: Neon PostgreSQL with pgvector
- AI: OpenAI API (GPT-4o, embeddings), HuggingFace
- Deployment: Vercel

## Build Principles
- Always run `npm install` after creating package.json
- Always fix errors automatically without asking
- After each feature, run the dev server to test
- Commit to GitHub after each major feature
- Keep all code clean and well-commented
- Prioritize demo-readiness above everything

## Key Features to Build
1. ✅ Authentication (JWT)
2. ✅ RAG-powered Course Chatbot
3. ✅ Homework Solver (Image → Solution)
4. ✅ Smart Tutor (Multi-subject)
5. ✅ AI Quiz Generator
6. ✅ Weak Area Analysis
7. ✅ Multilingual (English/Urdu)
8. ✅ Teacher & Student Dashboards

## Demo Must-Haves
- Clean, modern dark UI
- Fast response times
- No crashes during demo
- Impressive AI capabilities
- Live deployment URL
