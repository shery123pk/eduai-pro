SLIDE 2 — THE PROBLEM
================================================================================

  Millions of students in Pakistan face these challenges every day:

  1. ONE-SIZE-FITS-ALL TEACHING
     Traditional classrooms cannot adapt to each student's learning style,
     pace, or understanding level.

  2. EXPENSIVE TUTORING
     Quality private tutoring is unaffordable for most families, creating
     a massive gap between students.

  3. NO PSYCHOLOGICAL INSIGHT
     No tools exist to understand how a student feels, what motivates them,
     or what their emotional learning state is.

  4. LANGUAGE BARRIER
     Most AI education tools only work in English. Pakistani students who
     think and learn in Urdu are left behind.

  5. UNKNOWN WEAK AREAS
     Students don't know exactly where they are struggling or how to
     improve systematically.

  6. LACK OF MOTIVATION TRACKING
     No system tracks student consistency, streaks, or emotional wellness
     over time.

SLIDE 4 — KEY FEATURES (DEEP DIVE)
================================================================================

  1. AI CHAT TUTOR (English & Pakistani Urdu)
     ─────────────────────────────────────────
     • Ask any academic question — get instant, detailed answers
     • Draws geometric shapes and diagrams on request
     • Supports full Pakistani Urdu (خوش آمدید not سواگت)
     • Powered by OpenAI GPT-4
     • Language toggle: English ↔ Urdu in one click

  2. PAT — PERSONAL ASSESSMENT TEST (FLAGSHIP INNOVATION)
     ───────────────────────────────────────────────────────
     A 5-step psychological assessment that understands the student:

     Step 1 → Activity Log      : What subjects and hours studied today
     Step 2 → Emotional Check   : Current feelings and confidence level
     Step 3 → Challenges        : Biggest difficulties and strengths
     Step 4 → Future Plans      : Next session plan and weekly goals
     Step 5 → AI Report         : Full psychological learning profile

     Report includes:
     • Learning Style (Visual / Auditory / Kinesthetic / Reading-Writing)
     • Motivation Score
     • Emotional Wellness Score
     • Focus Level Score
     • Consistency Score
     • Personalized Recommendations
     • Next Session Study Plan
     • Encouragement Message

  3. MASTERY COACH (Bloom's Taxonomy)
     ────────────────────────────────
     5 progressive learning levels:
     Level 1 → Remember   : Recall facts and basic concepts
     Level 2 → Understand : Explain ideas in own words
     Level 3 → Apply      : Use knowledge in new situations
     Level 4 → Analyze    : Draw connections between ideas
     Level 5 → Evaluate   : Justify decisions and arguments

     • AI generates 5 new questions per level
     • Must score 60%+ to advance
     • Mastery Badge awarded on completing all 5 levels

  4. AI HOMEWORK SOLVER
     ────────────────────
     • Upload or describe homework problem
     • AI provides step-by-step solution
     • Subject name printed as bold heading (not "AI Solution")
     • Export to PDF or Print with professional formatting
     • Works in English and Urdu

  5. AI QUIZ GENERATOR
     ──────────────────
     • Enter a topic or upload a photo of notes
     • AI generates Multiple Choice Questions with 4 options
     • Correct answers highlighted after submission
     • Performance tracked for weak area analysis

  6. VIDEO LIBRARY
     ───────────────
     • 27 pre-loaded educational videos across 6 subjects
     • Mathematics, Physics, Chemistry, Biology, English, Entrepreneurship
     • YouTube integration with thumbnail previews
     • Admin can add new videos from any YouTube channel

  7. AI TOOLS HUB (7 Tools)
     ──────────────────────
     Tool 1 → Concept Explainer    : Clear explanations with analogies
     Tool 2 → Flashcard Generator  : Study flashcards from any topic
     Tool 3 → Worksheet Generator  : Practice at Easy/Medium/Hard level
     Tool 4 → Worksheet Animator   : Upload worksheet → visual lesson
     Tool 5 → Lesson Planner       : 7 teaching frameworks supported
     Tool 6 → Doubt Solver         : Step-by-step problem solutions
     Tool 7 → Topic Summarizer     : Concise topic summaries

  8. GAMIFICATION SYSTEM
     ──────────────────────
     • XP Points — earn experience for every activity
     • Levels — rank up as you learn
     • Streaks — daily study streak tracking
     • Badges — achievements for milestones
     • Daily Challenges — fresh tasks every day
     • Leaderboard — compete with other students

SLIDE 5 — BILINGUAL SUPPORT (UNIQUE TO PAKISTAN)
================================================================================

  WHY THIS MATTERS
  ─────────────────
  Most AI education tools worldwide are English-only.
  In Pakistan, millions of students think, read, and learn in Urdu.
  We built full Pakistani Urdu support — not just translation.

  WHAT WE BUILT
  ─────────────
  • Jameel Noori Nastaleeq font for authentic Urdu display
  • Right-to-left text rendering
  • Pakistani Urdu vocabulary (خوش آمدید, براہ کرم — not Indian Hindi words)
  • AI system prompts trained to respond in Pakistani Urdu only
  • Language toggle on every feature: English ↔ Urdu

SLIDE 6 — TECHNOLOGY STACK
================================================================================

  FRONTEND
  ─────────
  • React 18 + Vite      — Fast, modern UI framework
  • Tailwind CSS         — Utility-first styling
  • React Router         — Client-side navigation
  • KaTeX                — Math formula rendering

  BACKEND
  ────────
  • Node.js + Express    — REST API server
  • OpenAI GPT-4o        — AI tutoring, quiz, PAT analysis
  • GPT-4o Vision        — Homework image analysis
  • JWT Authentication   — Secure token-based auth
  • Neon PostgreSQL       — Cloud database

  AI CAPABILITIES
  ────────────────
  • GPT-4o Vision for homework image → solution
  • GPT-4 for chat tutoring and Q&A
  • AI quiz generation from topic or image
  • Psychological profiling (PAT) with JSON-structured output
  • Concept explanation, flashcard generation, lesson planning
  • All with bilingual English / Pakistani Urdu support

  INTEGRATIONS
  ─────────────
  • YouTube API          — Video library thumbnails and embeds
  • Google Fonts         — Jameel Noori Nastaleeq Urdu font
  • Browser Print API    — PDF export for homework solutions
