## 🚀 Deploy in 7 Easy Steps

### **📖 FOLLOW: DEPLOY_NOW.md** ← **Start Here!**

Quick overview:

1. **Create GitHub Repo** (2 min)
   - Go to: https://github.com/new
   - Name: **eduai-pro**
   - Click Create

2. **Push Code** (1 min)
   ```bash
   git remote add origin https://github.com/shery123pk/eduai-pro.git
   git push -u origin main
   ```

3. **Get API Keys** (15 min)
   - OpenAI: https://platform.openai.com/api-keys
   - HuggingFace: https://huggingface.co/settings/tokens
   - Neon: https://neon.tech

4. **Deploy Backend** (5 min)
   ```bash
   vercel login
   cd backend
   vercel --prod
   ```

5. **Add Environment Variables** (5 min)
   - Go to Vercel dashboard
   - Add API keys
   - Redeploy

6. **Deploy Frontend** (5 min)
   ```bash
   cd frontend
   npm run build
   vercel --prod
   ```

7. **Test & Celebrate!** 🎉

**Total Time: ~30 minutes**
